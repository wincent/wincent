# Appendix: How these screenshots were made

1.  Zoom the terminal to occupy the full screen.
2.  Adjust kitty font size so that there is no gap at top or bottom of window.
3.  Execute:

    ```sh
    for COLOR in $(color ls); do
        color $COLOR
        screencapture -T 1 -R 0,32,3024,1932 media/colorschemes/$COLOR.png & nvim -c 'call timer_start(2000, { -> execute("q") })' fig/fs/stat.ts
    done
    ```
